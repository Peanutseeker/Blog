HTML= HyperText Markup Language， 属于网页编程的核心内容

### elements in HTML

`HTML` 代码由元素组成，通常以 `\<xxx>  ... \</xxx>` 为一个层
可以用 <keyboard> F12</keyboard> 来查看网页的具体代码

HTML代码如果不加 elements 的话是会以正常形式直接显示在网页上的

具体讲解元素类：

| 元素    | 功能                      |
| ----- | ----------------------- |
| \<h1> | 标题，1 可以改为2-6，数字越大代表标题越小 |
| \<p>  | 段落                      |
